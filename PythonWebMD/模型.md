# 模型

- MVT设计模式中的model,专门负责和数据库交互对应(models.py)
- 由于Model中内嵌了ORM框架，所以不需要直接面向数据库编程

- 模型迁移(建表)
  - 生成迁移文件: 根据模型类生成创建表的语句
    - python manage.py makemigrations
  - 执行迁移文件: 根据模型类生成创建表的语句在数据库中创建表
    - python manage.py migrate
