# 关联查询

```
查询书籍为1的所有人物信息
查询人物为1的书籍信息
```

- 一对多

  - 一对应的模型类对象 多对应的模型类小写_set

  - - 

      ```
      例
      
      book = BookInfo.objects.get(id=1)
      book.peopleinfo_set.all()
      ```

      

- 多对一

  - 多对应的模型类对象. 

  - ```
    Peopleoinfo.objects.get(id=1)

    >>> person = PeopleInfo.objects.get(id=1)
